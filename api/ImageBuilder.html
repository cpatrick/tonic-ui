ImageBuilder

ImageBuilder is a React component using the AbstractMenu to encapsulate a ParameterSet. Not to be confused with tonic-core

The widget is imported using the following:

var ImageBuilder = require('tonic-ui/lib/react/viewer/ImageBuilder');

Table of Contents #

  • Properties

Properties #

  • queryDataModel: Instance of the QueryDataModel, used core data handler.

We expect data from the queryDataModel to be named 'image'.
