Probe3dViewer

Probe3dViewer is a React component using the AbstractMenu to encapsulate a ParameterSet with a set of controls regarding the probe locationand the orientation of the view to render.

The widget is imported with:

var Probe3dViewer = require('tonic-ui/lib/react/viewer/Probe3D');

Table of Contents #

  • InitialState
  • Properties

InitialState #

{
    renderMethod: 'renderXY',
    renderMethods: ['renderXY', 'renderZY', 'renderXZ'],
    probe: [
        this.props.imageBuilder.getProbe()[0],
        this.props.imageBuilder.getProbe()[1],
        this.props.imageBuilder.getProbe()[2]
    ]
}

Properties #

  • queryDataModel: Instance of the QueryDataModel to be used as core data handler.
  • imageBuilder: Instance of an image builder to be used with the current dataset.

We expect to call the following set of methods and properties on the imageBuilder instance:

  • setPipelineQuery(String)
  • setPushMethodAsBuffer()
  • setProbe(i, j, k)
  • getProbe() => [i,j,k]
  • metadata.dimensions
